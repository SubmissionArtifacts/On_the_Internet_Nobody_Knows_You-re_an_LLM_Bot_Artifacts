## Passive Traffic Analysis

This directory contains the passive traffic collected before and after the active experiments, spanning the period from **January 8, 2026** to **May 8, 2026**, as well as the notebook used to reproduce the passive traffic analyses presented in Appendix E.

The collected traffic primarily consists of visits from web crawlers, search engine bots, AI crawlers, and security scanning services.

> **Note.** Fingerprints may vary across browser versions, operating systems, network stacks, and software updates. Consequently, the fingerprints included in this artifact should be considered representative of the experimental period rather than permanent identifiers.
