## Scripts

The scripts provided in this repository illustrate the methodology used to evaluate browser automation frameworks, Crawl4AI and BrowserUse against our honeysites.

These scripts are not intended to provide strict, out-of-the-box reproducibility. To execute them successfully, users must adapt several configuration parameters, including:

* The target domain names (currently replaced with placeholders);
* The identifiers and selectors used by the honeysites' forms;
* Local deployment settings and credentials, when applicable.

The scripts can be made fully functional when used with the honeysite infrastructure provided in the `Honeysites/` directory and after updating the configuration to match the deployed environment.
